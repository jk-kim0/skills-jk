# 계획 문서 템플릿

이 템플릿은 예전 planning 전용 산출물 형식이다. 승인된 case에는 사용하지 않는다.

승인된 case의 계획 결과는 아래 문서를 사용한다.

- `README.md`
- `metadata.yaml`
- `evidence.md`
- `plan/result.md`
- `plan/handoff.md`

새 산출물은 다음 템플릿을 사용한다.

- `case-readme.md`
- `case-metadata.yaml`
- `case-evidence.md`
- `stage-result.md`
- `stage-handoff.md`
