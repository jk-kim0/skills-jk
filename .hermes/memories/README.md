# Hermes Memories

Store durable Hermes agent memory here.

These files are intentionally tracked by Git. Review changes before committing, and keep runtime session data in `.hermes/sessions/`.

Do not store secret values here. Non-sensitive references such as environment variable names, `.env` key names, config key names, 1Password vault names, 1Password item titles, 1Password secret references, and secret manager reference names are allowed.
