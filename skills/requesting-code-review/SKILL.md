# Requesting Code Review

## 목적

리뷰어가 빠르게 정확한 판단을 하도록 정보를 구조화합니다.

## 포함 항목

1. 변경 목적과 범위
2. 핵심 파일/로직 요약
3. 테스트 및 검증 결과
4. 위험 요소와 확인 요청 포인트

## 금지

- 검증 없이 "완료" 주장
- 변경 범위 불명확한 요청
