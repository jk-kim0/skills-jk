# Systematic Debugging

## 목적

추측이 아닌 근거로 원인을 찾아 수정합니다.

## 5단계

1. 재현 조건 고정
2. 관측 지점 추가 (로그/메트릭)
3. 원인 가설 최소 2개 설정
4. 실험으로 가설 검증
5. 수정 후 재현 케이스로 검증

## 산출물

- 재현 방법
- 근본 원인
- 수정 내용
- 재발 방지책
