# Verification Before Completion

## 목적

완료 선언 전에 결과를 증거 기반으로 검증합니다.

## 체크리스트

1. 요구사항 기준 검증
2. 자동 테스트 실행
3. 수동 검증 시나리오 실행
4. 성능/보안 회귀 여부 확인
5. 남은 리스크 명시

## 출력 형식

- 검증 명령
- 결과 요약
- 미해결 리스크
