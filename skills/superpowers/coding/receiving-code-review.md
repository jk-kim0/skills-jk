# Receiving Code Review

## 목적

리뷰 피드백을 빠르게 반영하고 결과를 명확히 공유합니다.

## 수행 절차

1. 코멘트 분류 (critical/warning/suggestion)
2. 재현 가능한 이슈부터 우선 수정
3. 수정 후 관련 테스트 재실행
4. 코멘트별 대응 결과 보고

## 대응 형식

- 코멘트 요지
- 조치 내용
- 검증 결과
- 잔여 이슈
