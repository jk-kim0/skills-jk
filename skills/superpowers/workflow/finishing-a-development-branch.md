# Finishing a Development Branch

## 목적

작업 브랜치를 PR 가능한 상태로 마무리합니다.

## 수행 절차

1. 변경 범위 재확인
2. 테스트/린트/빌드 실행
3. 리스크 및 후속 과제 정리
4. 커밋/PR 준비 상태 보고

## 종료 체크리스트

- 테스트 통과
- 계획 대비 누락 없음
- 문서/주석 업데이트 완료
- 알려진 리스크 명시
