# Writing Skills

## 목적

Codex가 정확히 트리거하고 실행할 수 있는 스킬 문서를 작성합니다.

## 작성 원칙

1. frontmatter의 `name`/`description`을 트리거 중심으로 작성
2. 절차는 재현 가능하고 짧게 유지
3. 긴 참고자료는 별도 파일로 분리
4. 출력 형식을 명시해 결과 일관성 확보

## 품질 기준

- 모호한 표현 없음
- 실행 순서 명확
- 성공/실패 판단 기준 포함
