## 8. 경쟁 분석 및 기회

### 8.1 현재 경쟁 상황

AIP 문서가 노출되는 검색어는 대부분 **일반 API 문서 검색**:
- "airtable docs" - Airtable 공식 문서와 경쟁
- "dify api" - Dify 공식 문서와 경쟁
- "slack api" - Slack 공식 문서와 경쟁

이러한 경쟁에서 순위 60-80위는 사실상 노출만 되고 클릭은 불가능.

### 8.2 차별화 키워드 기회

| 키워드 유형 | 예시 | 경쟁도 | 기회 |
|-------------|------|--------|------|
| MCP 통합 | "airtable mcp", "dify mcp integration" | 낮음 | 높음 |
| QueryPie 브랜드 | "querypie aip docs" | 없음 | 높음 |
| AI 에이전트 도구 | "ai agent airtable", "mcp server setup" | 낮음 | 높음 |
| 보안 키워드 | "secure mcp", "mcp guardrails docs" | 낮음 | 중간 |

### 8.3 권장 키워드 전략

**포기해야 할 키워드:**
- "airtable docs", "dify api", "slack api" 등 공식 문서 키워드
- 이들은 공식 사이트가 1위를 차지하며 경쟁 불가

**집중해야 할 키워드:**
- "MCP" 관련: "airtable mcp server", "dify mcp integration"
- "QueryPie" 브랜드: "querypie aip", "querypie mcp"
- "AI Agent" 관련: "ai agent database access", "mcp tools"

## 11. 결론

### 현재 상태 평가

| 영역 | 상태 | 평가 |
|------|------|------|
| 성장 추세 | 급성장 | ✅ 1월 15배 클릭 증가 |
| 한국 시장 | 우수 | ✅ CTR 22.7%, 순위 5.2위 |
| 일본 시장 | 양호 | ✅ CTR 7.2%, 순위 9.5위 |
| 미국 시장 | 문제 | ❌ 노출 1,330회, 클릭 0회 |
| 키워드 전략 | 개선 필요 | ⚠️ 일반 API 문서와 경쟁 중 |
| 인덱싱 | 양호 | ✅ 주요 페이지 인덱싱됨 |

### 핵심 발견사항

1. **급성장 중**: 1월 클릭 15배 증가, CTR 4.7배 개선
2. **한국/일본 집중**: 전체 클릭의 97%가 한국+일본에서 발생
3. **미국 기회 손실**: 노출 1,330회 중 0 클릭 (경쟁 키워드 문제)
4. **키워드 차별화 필요**: "MCP" 키워드로 차별화 필요

### 핵심 액션

1. **키워드 전략 전환**: 일반 API 문서 → MCP 통합 문서로 포지셔닝
2. **브랜드 키워드 강화**: "QueryPie AIP" 브랜드 노출 증가
3. **한국/일본 강화**: 고CTR 시장에서 콘텐츠 확대

### 예상 결과

권장 조치 실행 시:
- **1개월**: 일일 클릭 1.5 → 5 (3.3배 증가)
